Simple • Quick • Guide

This is a curated list to help you quickly get started with UI Design.

All software hardware learn resources jobs

Figma

The best tool to create UI designs, vector graphics, build prototypes, handoff projects to developers, share and track feedback. It's Free to get started.

Photoshop Elements

What you will need to edit and optimize images. It's Cost: $99, no subscription.

Macboor Air 15" M2

The best option for most designers (8GB, 256GB). It's fast, beautiful Retina XDR display with more screen real estate, longer battery life. Light (weights about 3.3lbs / 1.5kg).

MX Ergo

A great ergonomic mouse for your wrist and for multitasking (with programmable buttons applicable to each software, and can toggle use between devices without disconnecting bluetooth).

Figma: How to Use

This is a full course that will show you how to use Figma to create designs, styleguides, ui components, prototypes, manage project feedback and handoff process.

Dribbble

This is a social networking website where you can find design inspirations for your projects.

UX Collective

This is a website where you can find UX projects and case studies.

Nielsen Norman Group

This website features data driven proven case studies, quite helpful to share examples when making UX proposals to team or clients.

Google Job Search

The best Job search directory. Tip: when searching for jobs use keywords (role: ux ui designer), (industry: you have prior experience working in), (software: you use, like Figma).

Glassdoor

A great source to look for jobs, read company reviews, and learn industry salaries.
